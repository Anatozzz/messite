Politique de confidentialité

1. Introduction

Le site web élégance respecte votre vie privée. Cette politique explique comment nous collectons, utilisons et protégeons vos données personnelles.

2. Données collectées

Lorsque vous remplissez notre formulaire de contact, nous collectons les informations suivantes :

  • Votre nom et prénom
  • Votre adresse email
  • Votre numéro de téléphone (facultatif)
  • Votre message

3. Utilisation des données

Vos données sont utilisées uniquement pour :

  • Vous répondre
  • Vous envoyer des informations sur nos services
  • Éventuellement, vous contacter pour des offres futures si vous l'avez accepté

4. Conservation des données

Les données sont conservées pendant une durée maximale de 3 ans à compter du dernier contact.

5. Partage des données

Nous ne partageons jamais vos données personnelles avec des tiers sans votre consentement, sauf obligation légale.

6. Vos droits

Conformément au RGPD, vous disposez des droits suivants :

  • Droit d'accès à vos données
  • Droit de rectification
  • Droit de suppression
  • Droit d'opposition
  • Droit à la portabilité des données

Pour exercer ces droits, vous pouvez nous contacter à l'adresse suivante : contactweblegance@gmail.com

7. Sécurité

Nous mettons en œuvre toutes les mesures techniques nécessaires pour protéger vos données personnelles contre toute perte, accès non autorisé ou divulgation.

8. Modifications

Nous nous réservons le droit de modifier cette politique. En cas de changement, vous en serez informé via le site.
